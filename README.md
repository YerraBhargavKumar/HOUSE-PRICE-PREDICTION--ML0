# House Price Prediction using Machine Learning

This project predicts house prices based on features like area, bedrooms, and bathrooms.

## Features
- Linear Regression model
- Data processing using Pandas
- Web interface using Flask
- Real-time price prediction

## How to Run

1. Install dependencies:
pip install -r requirements.txt

2. Train model:
python train.py

3. Run app:
python app.py

## Technologies Used
- Python
- Flask
- Scikit-learn
- Pandas
- HTML/CSS
